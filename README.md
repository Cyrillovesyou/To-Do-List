# To-Do List Website

A simple and user-friendly To-Do List web application built with **HTML**, **CSS**, and **JavaScript**.  
It helps users create, manage, and track daily tasks easily.



##  Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and responsive design
- Tasks persist during the session (or using localStorage, if implemented)



##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
